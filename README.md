🧠 Resume Skill Gap Analyzer
An end-to-end Machine Learning + NLP project that analyzes resumes, predicts job roles, and identifies skill gaps with actionable suggestions.

🚀 Features
📄 Upload Resume (PDF)
🧠 Predict Job Role using Machine Learning
🧾 Extract Skills from Resume
📊 Calculate Match Score (TF-IDF + Cosine Similarity)
⚠️ Identify Missing Skills
💡 Suggest improvements

🏗️ Tech Stack
👨‍💻 Machine Learning
Python

scikit-learn

NLP (Text Cleaning, TF-IDF, Cosine Similarity)

🌐 Backend
Flask

📄 Resume Parsing
PyMuPDF

🎨 Frontend
HTML, CSS, JavaScript
